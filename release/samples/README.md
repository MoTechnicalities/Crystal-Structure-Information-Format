# Release Showcase Sample

This sample is designed for GitHub release packaging and demos.

## File

- `csif_release_showcase.py`

## What it demonstrates

1. Coherent vs contradictory phase separation.
2. Path-level auditable contradiction trace output.
3. Tiny cross-language bridge alignment (English to Spanish).

## Run

From repository root:

```bash
python3 release/samples/csif_release_showcase.py
```

## Expected result

Output includes:
1. A JSON block with metrics, checks, and audit trace.
2. Final line: `RESULT: PASS`.

## Inclusion guidance for GitHub release

Link this sample in release notes under "Try it in 10 seconds" and include the single run command above.
