# Public Roadmap (Release-Oriented)

This roadmap translates CSIF from a strong technical concept package into a durable open-source program.

## Track A: Scientific Credibility

1. Expand independent-source evaluation sets with strict protocol gates.
2. Publish error taxonomy for false positives and false negatives.
3. Add ablation studies separating phase-geometry gains from lexical heuristics.

## Track B: Product Reliability

1. Stabilize reproducibility scripts and expected-output checks.
2. Add lightweight CI for script execution and doc-link checks.
3. Add versioned changelog and release cadence.

## Track C: Open-Source Governance

1. Formalize contribution policy and review standards.
2. Add security reporting workflow.
3. Publish coding and documentation conventions for external contributors.

## Track D: World-Language Crystal Lobe Cluster (Vision Track)

Long-term objective:
- A federated cluster of auditable language banks covering world languages, with bridge mappings and explicit phase offsets.

Milestone D1 - Seed multilingual banks:
1. Build seed banks for at least 10 language families.
2. Define canonical concept anchors shared across banks.
3. Add provenance schema for language-specific evidence.

Milestone D2 - Cross-language bridge protocol:
1. Define bridge-term confidence and conflict resolution rules.
2. Add translation-independent proposition keys for evaluation.
3. Validate cross-bank contradiction tracing on multilingual claim pairs.

Milestone D3 - Audited federation at scale:
1. Add per-language and per-domain coverage metrics.
2. Add adjudication workflows for ambiguous bridge mappings.
3. Publish reproducible multilingual pilot benchmark reports.

Milestone D4 - Public demonstrator:
1. Query one proposition across multiple languages.
2. Return aligned traces and residual conflicts.
3. Show full provenance lineage in one report.
